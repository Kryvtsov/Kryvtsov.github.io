День 1. Мистер Безье.

transition-timing-function свойства:

  • ease - медленно - быстро - медленно (по умолчанию)
  • linear - действует одинаково на протяжении всей анимации.
  • ease-in - медленно - быстро
  • ease-out - быстро - медленно
  • ease-in-out - медленно - медленно
  • step-start - без анимации сразу применяет конечные стили.
  • step-end - без анимации применяет стили в конце
  • steps - указывает конкретное количество шагов для анимации.
  • cubic-bezier - задает анимацию по кривой Безье.
transition: width 3s ease;
transition: width 3s linear;
transition: width 3s ease-in;
transition: width 3s ease-out;
transition: width 3s ease-in-out;
transition: width 3s step-start;
transition: width 3s step-end;
transition: width 3s steps(5);
cubic-bezier(.05, .98, .94, .06);
cubic-bezier(.04,1.59,.56,-0.62);
cubic-bezier(.13,-0.75,.62,-0.58);
cubic-bezier(.04,1.58,.92,1.41);

На этом сайте можно сгенерировать собственную кривую Безье: http://cubic-bezier.com/

День 3. Занимацкий CSS.

CSS3 поддерживает покадровую анимацию, которая позволяет создавать сложные анимации объектов.

Задается она при помощи свойства - animation и правила @keyframes

При помощи свойства мы вызываем анимацию, а в правиле мы его задаем и описываем как оно должно работать.

Парочка советов:

1.Что бы в конце не было прыжка нужно делать последний слайд такой же как и оригинальное состояние объекта.

2.Для возврата исходного состояния можно указать пустые фигурные скобки.

3.Можно удалить одинаковые свойства которые находятся в промежуточных кадрах.

4.Анимация не должна двигать другие элементы сайта, поэтому нужно готовить для нее плацдарм.

5.Если вы хотите двигать объекты - используйте позиционирование.

День 4. Анатомия анимации.

Свойство animation имеет много различных настроек для анимации объектов.

Разбор анимации:

animation-name - задает имя для анимации.

animation-duration - задает время анимации.

animation-timing-function - кривая Безье которая задает стиль анимации.

animation-delay - задержка.

animation-iteration-count - счетчик (только целые числа), infinite - бесконечно.

animation-direction - указывает как должна выполняться анимация.

  • normal - анимация работает от начала и до конца
  • reverse - анимация работает от конца и до начала.
  • alternate - анимация работает от начала, до конца и до начала.
  • alternate-reverse - анимация рабоатет наоборот от конца в начало и конец.

Альтернативная анимация будет рабоать только если указан повтор больше 1.

animation-fill-mode - определяет как будет анимация воздействовать на элемент до и после анимации.

  • none - после анимации элемент возвращается обратно.
  • forwards - после анимации элемент застывает в последнем кадре.
  • backwards - анимация начинает работать до начала анимации.
  • both - начинает работать сразу и застывает в последнем положении.
animation: test 2s ease-in 0.5s 2 alternate both;

animation-play-state - отвечает за паузу. running - анимация работает; paused - пауза

Для задания анимации со всеми настройками необходимо соблюдать последовательность. Можно пропускать настройки, но соблюдать порядок.

День 5. Подготовка к flex'y.

RESET CSS - сброс стилей

Полностью обнуляет все стили по умолчанию у браузеров. Убирает отступы, поля, маркеры, межстрочные элементы и остальные оформленные по умолчанию у браузеров.

NORMALIZE CSS - нормализация стилей

В отличии от сброса стилей приводит все элементы сайта к единому нормальному виду во всех браузерах.
